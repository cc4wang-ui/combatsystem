## 📱 使用說明

### Dashboard Tab
- 查看本週任務進度
- 快速輸入今日完成事項
- 調整年度進度（點擊進度條可編輯）

### 日曆 Tab
- 點擊日期查看/編輯當天記錄
- 綠色圓點 = 該日有記錄
- 下方可嵌入 Google 日曆

### 任務管理 Tab
- 新增任務：點擊右上角 + 號
- 編輯任務：點擊任務文字
- 刪除任務：hover 後點擊垃圾桶
- 修改標籤：hover 後可選擇

### AI 總結 Tab
- 點擊「產生總結」
- AI 會分析你本週的：
  - 完成/未完成任務
  - 每日能量和壓力
  - 提供下週行動建議

## 🔗 Google 日曆設定

1. 打開 [Google 日曆](https://calendar.google.com)
2. 左側找到你的日曆 → 點擊「⋮」 → 「設定和共用」
3. 滾動到「整合日曆」區塊
4. 複製「日曆 ID」（通常是你的 email）
5. 在網站的「日曆」Tab → 點擊設定齒輪 → 貼上

**注意：** 日曆必須設為「公開」才能嵌入顯示

## 💾 資料說明

資料儲存在瀏覽器的 localStorage：

| Key | 內容 |
|-----|------|
| `cc-tasks` | 本週任務列表 |
| `cc-progress` | 年度進度數據 |
| `cc-daily-logs` | 每日記錄 |
| `cc-calendar-id` | Google 日曆 ID |

**⚠️ 注意事項：**
- 資料只存在你的瀏覽器（不同設備不會同步）
- 清除瀏覽器資料會清空記錄
- 建議定期用 AI 總結功能回顧
